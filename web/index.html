about

GlubLM is a 35-million-parameter transformer that pretends to be a goldfish. It has a hard 96-token context window, so it literally forgets everything after a sentence or two. The model runs 100% in your browser via ONNX Runtime Web - no server, no data leaves your device.

Inspired by GuppyLM and Ted Lasso's "be a goldfish" philosophy. Code and training dataset are on GitHub.
